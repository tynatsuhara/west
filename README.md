## Controls
Keyboard/PS4
* E/X - Drag Body/Interact
* F/Triangle release - Draw/Hide Weapon
* F/Triangle hold - Shout
* Q/Circle - Throw bag
* R/Square - Reload
* Mouse0/R2 - Shoot
* Mouse1/L2 - Melee
* P/RB - Change perspective
* MouseWheel/D-Pad Up+Down - Zoom
* Z+C/D-Pad Left+Right - Rotate Camera
* ESC/Options - Pause
